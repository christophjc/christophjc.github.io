Front-End Web Design Course

A free program by the New York Public Library.

Learn the foundations of coding and web design through

this 10-week introductory course in HTML and CSS.

What you'll learn

Phase 1 - HTML & CSS

Master the fundamentals of structuring and styling a website using HTML and CSS. Gain an understanding of layout principles, design concepts and best practices within the industry. Collaborate with peers and build your own portfolio-ready web project.

Syllabus

Week 1: HTML Fundamentals

Weeks 2-3: HTML Structure

Week 4: CSS Syntax

Week 5: Assigning classes and IDs

Week 6: CSS Positioning

Weeks 7-10: Collaboration Project
